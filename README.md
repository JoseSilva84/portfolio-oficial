# 💼 Portfólio - Dev José Silva

Um portfólio moderno e responsivo construído com HTML, CSS e JavaScript, apresentando projetos, especialidades e formação profissional de um desenvolvedor full stack.

## 🎯 Características Principais

✨ **Design Moderno**
- Interface com efeito de vidro (glassmorphism)
- Gradientes e animações suaves
- Cores vibrantes com tema verde e preto

📱 **Responsivo**
- Design mobile-first
- Menu hamburguer para dispositivos móveis
- Otimizado para todos os tamanhos de tela

🎨 **Funcionalidades Interativas**
- Revelação de elementos ao scroll (Scroll Reveal)
- Tooltips ao passar o mouse nos botões de especialidades e redes sociais
- Modal para visualizar currículo em PDF
- Efeitos de hover nos cards

🔗 **Integração com Redes Sociais**
- Links para Instagram, YouTube, LinkedIn e GitHub
- Botão de currículo que abre em modal

## 🚀 Como Usar

### Instalação
1. Clone ou baixe este repositório
2. Certifique-se de ter as imagens na pasta `/images`
3. Abra `index.html` em um navegador moderno

### Configuração
Para personalizar o portfólio, edite:

- **Dados pessais**: Altere o nome, descrição e links nas respectivas seções
- **Imagens**: Substitua as imagens em `/images` pelas suas próprias
- **Cores**: Modifique as cores no `style.css` (verde: `#00FF08`)
- **Links**: Atualize URLs de redes sociais, email e currículo
- **Conteúdo**: Atualize especialidades, formação e projetos conforme necessário

## 📱 Responsividade

O portfólio é totalmente responsivo com breakpoints em:
- **Desktop**: Acima de 1020px
- **Mobile**: Até 1020px
- **Tablet**: 768px a 1020px

## ✨ Animações e Efeitos

- **Scroll Reveal**: Elementos aparecem com animação ao entrar na tela
- **Hover Effects**: Cards de especialidades crescem e ganham sombra
- **Tooltips**: Aparecem ao passar o mouse com animação fade-in
- **Menu Mobile**: Abre do lado direito com overlay
- **Modal CV**: Abre com efeito de escala e blur do fundo
- **Animação Flutuante**: Imagem do topo flutuua suavemente

## 🎯 Funcionalidades JavaScript

### Arquivo `menu.js`:
1. **Menu Mobile** - Abrir/fechar menu hamburguer
2. **Scroll Header** - Adiciona classe ao header ao fazer scroll
3. **Scroll Reveal** - Observer para animar elementos ao scroll
4. **Modal CV** - Abre/fecha modal com currículo
5. **WhatsApp Integration** - Envia mensagem via WhatsApp
